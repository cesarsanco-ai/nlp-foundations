## 1. NER

Etiquetado secuencial **BIO/BILOU**; métricas por entidad: precisión/recall/F1 **por tipo**.

---

## 2. QA extractivo

Span start/end sobre pasaje; pérdida conjunta o dos cabezas.

---

## 3. Resumen

Extractivo (selección de oraciones) vs abstractivo (generación); **ROUGE** para extractivo, **BERTScore** / humano para abstractivo.

---

## 4. Modelos

Encoder (BERT) + CRF opcional para NER; T5/BART para resumen generativo.
